## [Unreleased]

### Added
- Initial release with five core commands
- `timeout` command for simulating delays with configurable exit codes
- `cpu-burn` command for CPU stress testing with parallelism control
- `mem-spike` command for memory allocation and pressure simulation
- `io-spam` command for disk I/O stress testing
- `exit` command for exit code testing
- Dry-run mode support across all commands
- Verbose logging option
- Rich terminal output using Spectre.Console
